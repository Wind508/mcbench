About McBench

McBench is a tool developed as part of the McLab project at McGill University. It aims to give insight about how Matlab programmers actually use Matlab. By performing structural queries against a large body of Matlab code, we can answer questions about how often certain features are used, or identify common patterns, which can help guide the development of compiler optimizations, virtual machines, and programmer tools (such as refactoring tools).

The application is written in Python, using the Flask web framework, together with the lxml toolkit for working with XML and XPath. It is developed and maintained by Ismail Badawi.
